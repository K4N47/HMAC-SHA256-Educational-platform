3. The HMAC Formula

HMAC(K, m) = H( (K'⊕opad) ∥ H( (K'⊕ipad) ∥ m ) )
H — SHA-256 hash function
K' — Key padded to 64 bytes
ipad — 0x36 × 64
opad — 0x5C × 64

4. Why Two Nested Hashes?

5. Key Normalization

Key ≤ 64 bytes
Pad with 0x00 to reach 64 bytes
K' = K ∥ (0x00 × (64-|K|))
Key > 64 bytes
Hash it first, then pad
K' = SHA-256(K) ∥ (0x00 × 32)
